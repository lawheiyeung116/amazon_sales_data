**amazon.csv**: a raw dataset taken from Kaggle: https://www.kaggle.com/datasets/karkavelrajaj/amazon-sales-dataset  
**docker-compose.yaml**: .yaml file used to set up the Docker container with MySQL image  
**mysql_db.py**: preprocess the raw dataset and insert data directly to the database with mysql.connector  
**sales_data_backup.sql**: the backup .sql file created by mysqldump command
